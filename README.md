# Smart Expense Tracker

A beginner-friendly Expense Tracker built using Python and Streamlit.

## Features
- Add daily expenses
- Auto category detection
- Light / Dark mode
- Monthly insights
- Spending trends with charts

## Tech Stack
- Python
- Streamlit
- Pandas

## How to Run
1. Install dependencies
   pip install streamlit pandas
2. Run app
   streamlit run app.py
